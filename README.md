## План проекта «Блокнот с подсветкой синтаксиса и темами оформления»

### 1. Интерфейс пользователя (основное окно)
- Многостраничный редактор (вкладки) для работы с несколькими файлами одновременно.
- Строка меню: Файл, Правка, Вид (темы, семантика), Справка.
- Панель инструментов с кнопками: Создать, Открыть, Сохранить, Копировать, Вставить, Поиск/Замена.
- Область редактирования с поддержкой прокрутки и отображения номеров строк.
- Строка состояния: текущий язык программирования, тема оформления, позиция курсора, кодировка файла.

### 2. Модуль поддержки семантики языков программирования
- Автоматическое определение языка программирования по расширению файла (`.py`, `.js`, `.html`, `.css`, `.cpp`, `.java` и др.).
- Подсветка синтаксиса для ключевых слов, строк, комментариев, чисел, операторов.
- Автодополнение кода (на основе текущего языка) — вызов по `Ctrl+Space`.
- Проверка парных скобок (`{}`, `[]`, `()`) и их подсветка при наведении.
- Возможность вручную переключить язык программирования через меню «Семантика».

### 3. Модуль тем оформления
- Набор встроенных тем: «Светлая», «Тёмная», «High Contrast», «Solarized Light», «Monokai».
- Смена темы оформления без перезапуска приложения.
- Настройка шрифта (тип, размер) отдельно от темы.
- Пользовательские темы: возможность редактировать цвета фона, текста, синтаксиса (через JSON-конфиг).
- Предпросмотр темы в отдельном окне до применения.

### 4. Файловый модуль
- Открытие/сохранение файлов в кодировках UTF-8, ANSI, UTF-8 with BOM.
- Поддержка «недавних файлов» (список до 10 элементов).
- Автосохранение каждые N минут (настраивается).
- Восстановление несохранённых вкладок при следующем запуске.

### 5. Модуль поиска и замены
- Поиск по текущему документу (с учётом регистра, целых слов, регулярных выражений).
- Замена одного вхождения или всех сразу.
- Подсветка всех найденных совпадений в документе.
- Поиск по всем открытым файлам (с отображением результатов в отдельной панели).

### 6. Модуль истории изменений (Undo/Redo)
- Неограниченная история изменений (или ограниченная настраиваемым числом шагов).
- Отмена (`Ctrl+Z`) и повтор (`Ctrl+Y`) любого действия.
- Отображение последних действий в выпадающем списке (опционально).

### 7. Модуль экспорта и печати
- Печать текущего документа с предпросмотром.
- Экспорт в HTML с сохранением подсветки синтаксиса и тем оформления.
- Экспорт в PDF (через виртуальный принтер или встроенную библиотеку).

### 8. Модуль дополнительных возможностей
- Подсветка текущей строки.
- Отображение невидимых символов (пробелы, табуляции, концы строк).
- Масштабирование текста (`Ctrl+колесо мыши` или `Ctrl+/Ctrl-`).
- Перенос длинных строк (wrap text) — включаемый режим.
- Закладки внутри документа (быстрая навигация).

### 9. Система настроек
- Сохранение настроек в конфигурационный файл (`settings.json`).
- Настройки:
  - Тема по умолчанию.
  - Язык программирования по умолчанию для новых файлов.
  - Шрифт, размер, интерлиньяж.
  - Автосохранение, интервал.
  - Показывать/скрывать номера строк, панель инструментов, строку состояния.
- Сброс настроек на стандартные.

### 10. Справочное руководство (встроенное)
- Краткое описание всех возможностей (F1 или меню Справка → Руководство).
- Примеры работы с подсветкой синтаксиса на разных языках.
- Описание горячих клавиш (полный список).
- Информация о версии и авторе.
